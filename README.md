# 📱 One UI 7 Simulator

A smooth and interactive **One UI 7 simulator**, inspired by Samsung's latest user interface. This project recreates the One UI 7 experience with realistic visuals, fluid transitions, and built-in apps.

🌐 **Live Demo:** [elpedr.github.io/1ui7adapt](https://elpedr.github.io/1ui7adapt)

## ✨ Included Apps

- 🧩 **Good Lock**
- 🏠 **Home Up**
- 🧮 **Calculator**
- ⚙️ **Settings**
- 📞 **Phone**
- ⏰ **Clock**
- 🖼️ **Gallery**
- 🎵 **Music**

## 🎮 Experience

- Fluid and responsive animations
- Realistic layout and transitions
- Touch-friendly interactions
- Built entirely in **Construct 2**

Feel free to explore, contribute, or suggest features!
